# 协同旅行项目

一个基于 Vue3 + TypeScript 的协同旅行管理平台，支持团队协作制定旅行计划、旅行记账分摊和社区分享功能。

## 功能特性

### 🗺️ 行程管理
- **创建行程**: 支持创建详细的旅行计划，包括目的地、时间、描述等
- **成员管理**: 邀请团队成员参与行程规划，支持不同角色权限
- **行程安排**: 详细的日程安排，包括交通、住宿、活动等
- **行程状态**: 计划中、进行中、已完成等状态管理

### 💰 记账分摊
- **账单管理**: 记录旅行过程中的各项支出
- **智能分摊**: 支持平均分摊、自定义金额、按比例分摊
- **分类统计**: 按交通、住宿、餐饮、活动等分类统计
- **结算管理**: 跟踪账单结算状态，支持部分结算

### 🌍 旅行社区
- **行程分享**: 将精彩行程分享到社区供他人参考
- **浏览发现**: 浏览其他用户分享的旅行经验
- **互动功能**: 点赞、收藏、评论等社交功能
- **标签筛选**: 通过标签快速找到感兴趣的内容

### 👤 个人中心
- **个人资料**: 管理个人信息和头像
- **我的行程**: 查看和管理所有创建的行程
- **我的分享**: 管理已分享到社区的内容
- **收藏夹**: 收藏喜欢的行程和内容
- **隐私设置**: 控制个人信息和行程的可见性

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI 组件库**: Element Plus
- **路由管理**: Vue Router 4
- **状态管理**: Pinia
- **构建工具**: Vite
- **HTTP 客户端**: Axios
- **日期处理**: Day.js

## 页面功能说明

### 仪表盘 (Dashboard)
- 显示行程和支出统计概览
- 快速操作入口
- 最近行程和账单列表

### 行程管理
- **行程列表**: 展示所有行程，支持筛选和搜索
- **创建行程**: 表单化创建新行程，支持成员邀请
- **行程详情**: 详细的行程信息，包括日程安排、成员管理、相关账单

### 记账管理
- **账单列表**: 展示所有账单，支持多维度筛选
- **添加账单**: 创建新账单，支持多种分摊方式
- **统计分析**: 支出统计和分析

### 旅行社区
- **社区动态**: 浏览所有公开分享的行程
- **详情页面**: 查看完整的行程安排和费用参考
- **分享功能**: 将自己的行程分享到社区

### 个人中心
- **个人资料**: 编辑个人信息
- **我的内容**: 管理行程、分享和收藏
- **账户设置**: 隐私设置和安全设置

## 数据模型

### 核心实体
- **User**: 用户信息
- **Trip**: 行程信息
- **ItineraryItem**: 行程安排项
- **Expense**: 账单信息
- **CommunityPost**: 社区分享

### 关系说明
- 一个行程可以有多个成员
- 一个行程可以有多个行程安排项
- 一个行程可以关联多个账单
- 一个账单可以有多个参与者和分摊记录

## 开发说明

### 组件设计原则
- 组件职责单一，便于维护
- 使用 TypeScript 确保类型安全
- 遵循 Vue 3 Composition API 最佳实践

### 样式规范
- 使用 Element Plus 组件库保持 UI 一致性
- 自定义样式使用 scoped CSS
- 响应式设计适配不同屏幕尺寸

### 状态管理
- 使用 Pinia 进行全局状态管理
- 页面级状态使用 Vue 3 响应式 API
